## 🇧🇷 Descrição

Este projeto foi desenvolvido como parte de um desafio prático com o objetivo de aplicar conceitos de **Programação Orientada a Objetos (POO)** em JavaScript.

O sistema representa um herói com atributos básicos e um método de ataque que varia conforme o tipo do personagem.

---

## 🚀 Tecnologias utilizadas

- JavaScript (ES6)

---

## 🧠 Conceitos aplicados

- Classes e Objetos
- Métodos
- Estruturas condicionais (if/else)
- Variáveis
- Template Strings

---

## ⚙️ Funcionalidades

O sistema permite:

- Criar um herói com:
  - Nome
  - Idade
  - Tipo (mago, guerreiro, monge, ninja)

- Executar um método de ataque que retorna uma mensagem personalizada de acordo com o tipo do herói.
